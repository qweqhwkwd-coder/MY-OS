## 00 · Дашборд `/today` ✅ (базово)
Главный экран: дата+приветствие, RPG-карточка (статы+уровень), кольца активности за день (ритуалы/питание/тренировка/вода), топ-3 задачи, ближайшая встреча, быстрые действия.
**Сейчас:** `/today` и `/api/today` отдают воду/ритуалы/задачи/ккал за день. Колец активности, топ-3 задач и ближайшей встречи на одном экране нет.

## 01 · Задачи 🟡 (ядро готово)
Создание (название, дедлайн, приоритет 🔴🟡🟢, тег), подзадачи, повторяющиеся, drag&drop приоритет, архив с поиском и статистикой. Быстрый ввод через бота.
**Сейчас:** добавление/список/выполнение/удаление по названию — через бота и `/api/tasks`. Подзадачи, повторяющиеся, drag&drop, архив — нет.
**RPG:** выполненная задача → +3 XP в Дисциплину (реализовано).

## 02 · Ритуалы ✅
Создание (название, иконка, время, дни недели), отметка да/нет, стрик-счётчик, тепловая карта истории. **Вода — встроенный виджет внутри ритуалов** (не отдельный пункт меню).
**RPG:** стрики → Дисциплина; вода → Здоровье.

## 03 · Встречи 🟡 (базово)
Добавление (название, дата/время, длительность, место, участники), статусы, архив по неделям, поиск, статистика.
**Сейчас:** `/addmeeting`, `/meetings` — название/дата/время, список ближайших. Длительность, место, участники, статусы, архив — нет.

## 04 · Тренировки + AI-тренер 🟡 (лог есть, AI-тренер нет)
Кастомные шаблоны активностей (быстрый тап) + свободный ввод + ручное создание. AI определяет категорию (кардио/силовое/смешанное/гибкость).
AI-тренер: атлетический профиль → физ-тесты → генерация программы с периодизацией → адаптация по фидбеку → учёт контекста (сон/питание/стресс).
**Сейчас:** `/workout` — свободный ввод, тип определяется по ключевым словам (не AI). Шаблонов, AI-тренера, программ — нет.
**RPG:** кардио → Выносливость; силовое → Сила (реализовано); соответствие программе → Дисциплина (нет программ).

## 05 · Питание 🟡 (ручной ввод есть, фото/AI нет)
Лог приёмов (завтрак/обед/ужин/перекус), 3 способа ввода: фото этикетки (Vision→КБЖУ), фото блюда, ручной ввод. AI-анализ приёма, дневной прогресс vs норма.
**Сейчас:** `/addfood` — название + опц. граммы + ккал, лог за день. Фото, AI-анализ, разбивка по приёмам — нет.
**RPG:** попадание в КБЖУ-норму → Питание (сейчас просто +2 XP за запись, без проверки нормы).

## 06 · Дневник ✅ (базово)
Свободная запись (текст/голос/фото), настроение 1–5, теги, тепловая карта, статистика настроения.
**Сейчас:** `/journal` (бот) и Mini App (минимальный экран — список последних 10 записей + додавання) — текст + опц. настроение 1–5. Архив по дате, редактирование/удаление записи (ни в боте, ни в Mini App), голос/фото, теги, тепловая карта — нет.
**RPG:** регулярные записи → Рефлексия (+2 XP за запись, реализовано).

## 07 · Цели ✅ (базово)
Создание (название, дедлайн, метрика, сфера жизни), подцели, привязка задач, прогресс-бар, мини-дневник цели.
**Сейчас:** `/addgoal`, `/goals`, `/goalsdone` — название + опц. дедлайн. Метрика, подцели, привязка задач — нет.
**RPG:** достижение цели → +10 XP Дисциплина + 5 XP Рефлексия (реализовано).

## 08 · Идеи ✅ (базово)
Быстрая запись (текст/голос/фото), теги, статусы (сырая/в работе/реализована/отложена), превращение идеи в задачу.
**Сейчас:** `/idea` — текст, список. Голос/фото, теги, смена статуса, превращение в задачу — нет.

## 09 · Дайджест 🟡 (по запросу, без AI и расписания)
Еженедельный AI-отчёт по всем модулям: цифры недели, что хорошо/провалено, динамика RPG, 3 рекомендации. Авто в воскресенье 20:00 + по запросу.
**Сейчас:** `/digest` и `/api/digest` — цифры за 7 дней по всем модулям, без AI-анализа и без авторассылки (нужен планировщик, см. `roadmap.md`).

## 10 · Профиль / RPG-статы 🟡 (статы+ранг+HP готовы, физпрофиль нет)
Физпрофиль (имя, фото, вес, рост, возраст, цели КБЖУ/воды). RPG: радарная диаграмма, ранги, прогресс. Подробно — `rpg-system.md`.
**Сейчас:** `/stats`, `/api/profile`, `ProfileModal` в Mini App — 8 статов, level, ранг, HP, стрик, XP за сегодня. Физпрофиль (фото/рост/возраст), радарная диаграмма — нет.

## 11 · Сон ✅ (базово)
Лог (отход+подъём→длительность авто), качество 1–5, заметки, графики, корреляции в дайджесте. Голосом утром.
**Сейчас:** `/sleep` — время засыпания/пробуждения → длительность авто, история. Качество, заметки, графики, голос — нет.
**RPG:** регулярный сон 7–9ч → Здоровье (+3 XP, реализовано). AI-тренера нет.

## 12 · Финансы ✅ (базово)
Расход (сумма, валюта, категория, заметка), доход опционально, бюджет по категориям, голосовой ввод, PieChart категорий.
**Сейчас:** `/spend`, `/finance` — сумма + категория, сводка за 7 дней по категориям текстом. Валюта (одна по умолчанию), доход, бюджеты, голос, графики — нет.
**RPG:** запись расхода → +1 XP Финансы (реализовано).

## 13 · Замеры тела ✅ (только вес)
Лог раз в 1–2 недели (вес, обхваты, % жира), фото прогресса приватно, напоминание, графики динамики.
**Сейчас:** `/weight` — вес, история последних замеров. Обхваты, % жира, фото, напоминания, графики — нет.
**RPG:** запись веса → +1 XP Рефлексия (реализовано).

## 14 · Inbox ✅ (базово)
Свалка-без-категории (текст/голос/фото), потом разбор (→ задача/идея/дневник/встреча/удалить), AI-предложение категории. Любое нераспознанное сообщение бота падает сюда.
**Сейчас:** catch-all кладёт текст в Inbox, `/inbox` показывает и отмечает обработанным. Голос/фото, разбор по категориям, AI-предложение — нет.

## 15 · Колесо баланса ✅ (базово)
Еженедельная самооценка 8 сфер (1–10 слайдером), радарная диаграмма, AI-анализ провалов, связь с целями. Напоминание в воскресенье.
**Сейчас:** `/balance` — 8 сфер по 1–10 через инлайн-кнопки бота (не слайдер). Радарная диаграмма, AI-анализ, напоминание по расписанию — нет.

## 16 · Умные уведомления ❌
Системный слой, не отдельный экран. Напоминания (время), контекстные (триггер по данным), мотивационные (достижения). Требует планировщика — Фаза 5.

## 17 · Настройки 🟡 (только тема)
Редактирование профилей, часовой пояс, язык (RU/UA/EN), управление уведомлениями, экспорт данных (JSON/CSV), тема, скрытие модулей из навигации.
**Сейчас:** вкладка настроек в `ProfileModal` (Mini App) — переключатель темы light/dark/auto. Профиль, часовой пояс, язык, уведомления, экспорт, скрытие модулей — нет.

**Сейчас:** отдельного welcome-экрана нет (удалён при редизайне «Паперова ОС» 02.07.2026; цитата дня — строкой на «Сьогодні»). Физпрофиль заполняется вручную во вкладке «Тіло» (`ProfileModal`), шаги онбординга 1–5 как flow — не реализованы.
