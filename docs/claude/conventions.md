# Конвенции, дизайн, долги

> Назад: [CLAUDE.md](../../CLAUDE.md)

## Код-стиль / процесс

- **Один процесс:** FastAPI + aiogram вместе. Все хендлеры в `backend/main.py`.
- **Вся работа с данными — в `backend/db.py`.** Не разбрасывать запросы по хендлерам.
- **Конфиг — только через `config.py`** (pydantic-settings). Не читать `os.environ` напрямую.
- **`@dp.message()` catch-all регистрируется ПОСЛЕДНИМ** — иначе перехватит команды.
- **Язык интерфейса — украинский, жёстко зашит** (и в боте, и в Mini App) —
  это целевой основной язык, подтверждено автором. Раньше здесь было написано
  «RU» — неверно. i18n не внедрён; RU/EN остаются в планах как дополнительные.
- **Автотестов нет.** Минимальная проверка: `python -m py_compile backend/*.py`.
- **Секреты — только в Render Environment.** Никогда в репо. `.env` в `.gitignore`.

## Дизайн-система Mini App

Целевая палитра — та, что уже реализована (решено: текущая реализация — это
теперь официальный план, старый план из «Фазы 2» снят). Ниже — фактические
значения из `miniapp/src/index.css`, ими и руководствоваться при дизайне
новых экранов.

**Цвета (CSS-переменные с `data-theme`):**
- светлая: `--bg #f8f7f4` · `--ink #1a1a1a` · `--muted #999999` · `--accent #4338ca` (индиго)
- тёмная: `--bg #141414` · `--ink #f0efec` · `--muted #666666` · `--accent #818cf8`
- HP-бар: `--hp-hi #16a34a` (зелёный) / `--hp-mid #d97706` (оранж) / `--hp-lo #dc2626` (красный)
- модульные акценты (редизайн «Паперова ОС», 02.07.2026): `--mod-water/-rituals/-food/-finance/-sleep/-workouts` — валидированы для обеих тем; на странице модуля цвет живёт ТОЛЬКО в 2px линейке слева и mono-барах, больше нигде
- Тема переключается в `ProfileModal` (light/dark/auto), `auto` слушает `prefers-color-scheme`.

**Язык дизайна «Паперова ОС» (действующий, с 02.07.2026):**
- прогресс — типографские mono-бары `▮▮▮▮▮▮▯▯▯▯` (`components/MonoBar.tsx`), НЕ графические полосы и НЕ `rounded-full`; скругления в системе вообще не используются
- навигация: сетка `NavGrid` только на главной; внутри модуля — `HubStrip` (⌂ + номер/имя + ‹ ›); закрытые модули — строкой «СКОРО: …», не плитками
- иконки-эмодзи в UI не используются (кроме 🔥 стрика и праздничных тостов-майлстоунов); кнопки — текстовые, mono
- pressed-состояние — класс `.press-invert` (инверсия в чернила), тактильность — `haptic()` из utils.ts; цели нажатия ≥44px
- XP-тосты — формат журнала `+5 XP → ДИСЦИПЛІНА`, инверсия (чернила/бумага)

Старый план (`#0A0A0A`/`#F5F5F5` фон, неоново-жёлтый `#E8FF00` акцент,
Space Grotesk/Inter/JetBrains Mono) — **отменён**, не использовать как ориентир
для новых экранов.

**Фактический стек фронта:** React 19 + Vite + Tailwind v4, обычный
`useState`/`useEffect`. Целевые библиотеки из старого плана (shadcn/ui,
Recharts, @dnd-kit/sortable, TanStack Query, Zustand, supabase-js) — НЕ внедрены.

1. Авторизация под одного юзера — `service_role` обходит RLS, моста initData→JWT нет.
2. Синхронный supabase-py в async блокирует event loop.
3. ~~Нет отмены воды~~ — есть undo последнего добавления (Water + quick-add).
4. Нет планировщика/уведомлений/дайджеста-по-расписанию (free-хостинг); `/digest` есть только по запросу.
5. Холодный старт Render free ~30–60с.
6. Нет автотестов.
7. Нет `stats_history` → нет графиков динамики.
8. Миграции вручную, строго по одной.
9. Catch-all `on_any` теперь кладёт сообщение в Inbox (не просто эхо-подсказка).
10. Жёсткий **украинский** без i18n (целевой язык подтверждён, RU/EN — план на будущее, не сейчас).
11. Mini App покрывает только 5 из 12+ реализованных модулей.

## Расхождения «продукт vs код» (держать в голове)

- Продукт: **5** RPG-статов. Код: **8** (источник правды — 8, см. `rpg-system.md`).
- Продукт упоминает Railway. Решение MVP: **Render** (Railway free только 30 дней).
- Продукт описывает AI с первого дня. MVP: AI отложен на Фазу 4, КБЖУ формулой.
- Продукт/доки предполагали русский интерфейс. Реальный и целевой — украинский (решено).
- Продукт/доки описывали дизайн-систему (монохром + неоново-жёлтый акцент,
  Space Grotesk/Inter/JetBrains Mono). Решено: целевая палитра — та, что уже
  реализована в Mini App (тёплый off-white/индиго, см. раздел выше), старый
  план снят.
