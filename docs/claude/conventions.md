## Код-стиль / процесс

- **Один процесс:** FastAPI + aiogram вместе. Все хендлеры в `backend/main.py`.
- **Вся работа с данными — в `backend/db.py`.** Не разбрасывать запросы по хендлерам.
- **Конфиг — только через `config.py`** (pydantic-settings). Не читать `os.environ` напрямую.
- **`@dp.message()` catch-all регистрируется ПОСЛЕДНИМ** — иначе перехватит команды.
- **Язык интерфейса — фактически украинский, жёстко зашит** (и в боте, и в Mini App).
  Раньше здесь было написано «RU» — неверно, сверено напрямую с текстами в коде.
  i18n не внедрён (планируется RU/UA/EN — решить, какой язык целевой основной).
- **Автотестов нет.** Минимальная проверка: `python -m py_compile backend/*.py`.
- **Секреты — только в Render Environment.** Никогда в репо. `.env` в `.gitignore`.

## Известные проблемы / TODO (актуальный список)

1. Авторизация под одного юзера — `service_role` обходит RLS, моста initData→JWT нет.
2. Синхронный supabase-py в async блокирует event loop.
3. Нет отмены воды.
4. Нет планировщика/уведомлений/дайджеста-по-расписанию (free-хостинг); `/digest` есть только по запросу.
5. Холодный старт Render free ~30–60с.
6. Нет автотестов.
7. Нет `stats_history` → нет графиков динамики.
8. Миграции вручную, строго по одной; `002_add_stats.sql` потерян на диске (см. `database.md`).
9. Catch-all `on_any` теперь кладёт сообщение в Inbox (не просто эхо-подсказка).
10. Жёсткий **украинский** без i18n (не русский, как было написано ранее).
11. Mini App покрывает только 5 из 12+ реализованных модулей.
12. Дизайн-система Mini App разъехалась с задокументированным планом (см. раздел выше).
