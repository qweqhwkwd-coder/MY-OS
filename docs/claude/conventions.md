# Конвенции, дизайн, долги

> Назад: [CLAUDE.md](../../CLAUDE.md)

## Код-стиль / процесс

- **Один процесс:** FastAPI + aiogram вместе. Все хендлеры в `backend/main.py`.
- **Вся работа с данными — в `backend/db.py`.** Не разбрасывать запросы по хендлерам.
- **Конфиг — только через `config.py`** (pydantic-settings). Не читать `os.environ` напрямую.
- **`@dp.message()` catch-all регистрируется ПОСЛЕДНИМ** — иначе перехватит команды.
- **Язык интерфейса — RU, жёстко зашит.** i18n не внедрён (планируется RU/UA/EN).
- **Автотестов нет.** Минимальная проверка: `python -m py_compile backend/*.py`.
- **Секреты — только в Render Environment.** Никогда в репо. `.env` в `.gitignore`.

## Дизайн-система Mini App (для Фазы 2)

**Стиль:** минимализм, строгость, монохром. Вдохновение — MEOS. Острые углы, без декора.

**Цвета:**
- `#0A0A0A` фон (тёмная) / `#F5F5F5` фон (светлая)
- `#FFFFFF` / `#000000` текст
- `#1A1A1A` / `#EBEBEB` карточки
- `#E8FF00` акцент (неоново-жёлтый) — RPG, прогресс, активные элементы
- `#666666` вторичный текст

**Шрифты:** Space Grotesk (заголовки) · Inter (body) · JetBrains Mono (цифры: КБЖУ, статы, XP).

**Компоненты:** карточки 0 border-radius · кнопки outline по умолчанию, filled только для primary · нумерация модулей 01,02… · прогресс-бары тонкие 2px акцентным цветом.

**Целевые библиотеки фронта:** shadcn/ui + Tailwind, Recharts (RadarChart/Line/Area/Bar/Pie), @dnd-kit/sortable, TanStack Query v5, Zustand, supabase-js.

## Известные проблемы / TODO (полный список)

1. Авторизация под одного юзера — `service_role` обходит RLS, моста initData→JWT нет.
2. Синхронный supabase-py в async блокирует event loop.
3. Фарм XP на ритуалах — нет guard «раз в день».
4. Нет отмены воды.
5. Нет планировщика/уведомлений/дайджеста (free-хостинг).
6. Холодный старт Render free ~30–60с.
7. Нет автотестов.
8. Нет `stats_history` → нет графиков динамики.
9. Миграции вручную, строго по одной.
10. `tasks` и `food_logs` — таблицы без кода.
11. Секрет вебхука выводится из токена (не отдельная переменная).
12. Catch-all `on_any` эхо-подсказка на нераспознанное.
13. Жёсткий RU без i18n.
14. Система рангов не реализована (только уровень = среднее/100).
