## Активные баги / долги (важно при доработке)

1. **Фарм XP на ритуалах.** `toggle_ritual` при снятии удаляет лог → повторная отметка снова даёт +2 XP. Нет guard «один раз в день». Допустимо для соло, чинить позже.
2. **Нет отмены воды.** `add_water` только прибавляет. XP за воду — один раз в день при первом пересечении цели (`before < goal <= total`).
3. **Нет `stats_history`** → нет графиков динамики статов.
4. **Нет автотестов.** Только `python -m py_compile`.
5. **Жёстко зашитый русский**, без i18n.
6. **Система рангов не реализована** — есть только уровень = среднее/100.
7. **Существовал черновик схемы на 27 таблиц** в заброшенном каталоге. Боевую БД сверять с `001` + `002`, не с ним.
