## 8 характеристик (текущая модель в коде)

> ⚠️ Расхождение с продуктовым документом: оригинал описывает **5** статов.
> MVP расширил до **8** (переменная `STATS` в `db.py`). Это решение зафиксировано в ROADMAP_v2 §2. Источник правды — 8.

| Стат (ключ) | Качается от | Статус |
|---|---|---|
| 💪 Сила (`strength`) | силовые тренировки | Фаза 4 |
| 🏃 Выносливость (`endurance`) | кардио | Фаза 4 |
| ❤️ Здоровье (`health`) | вода ✅, сон | вода работает |
| 🥗 Питание (`nutrition`) | еда (КБЖУ) | Этап 4 |
| 🧠 Дисциплина (`discipline`) | ритуалы ✅, задачи в срок | ритуалы работают |
| 🪞 Рефлексия (`reflection`) | дневник, цели, медитация | Фаза 3 |
| 💰 Финансы (`finance`) | расходы, бюджет | Фаза 3 |
| 📚 Интеллект (`intellect`) | чтение, учёба | Фаза 3 |

## Уровень

```
level = floor(среднее_XP_по_8_статам / 100)
```

Максимум XP одной характеристики: 5000.

## Начисление XP — реализовано сейчас

- Вода → `health` +2 (один раз в день при первом пересечении дневной цели).
- Ритуал отмечен → `discipline` +2.

Все начисления идут через `add_xp` и (по дизайну) логируются в `xp_events`.

## Начисление XP — целевые правила (из продукта, не все внедрены)

```
СИЛА:        +5 силовая трен · +2 за 10кг объёма · +3 личный рекорд
ВЫНОСЛИВОСТЬ: +5 кардио · +3 за 5км · +3 за 500м бассейн · +4 трен по программе
ПИТАНИЕ:     +5 день в норме ±10% ккал · +3 норма белка · +2 норма воды · −3 пропуск
ДИСЦИПЛИНА:  +2 ритуал · +5 стрик 7/7 · +3 задача в срок · −2 просрочка >2дн · +2 разбор Inbox · +3 сон 7-9ч
РЕФЛЕКСИЯ:   +4 запись дневника · +5 запись с настроением · +10 достигнута цель · +3 колесо баланса · +2 замеры · +1 расход (макс 3/день)
```

## Ранги ❌ (не реализованы — есть только уровень)

**Общий ранг** — по среднему XP всех статов:
Росток (0–200) · Камень (201–500) · Железо (501–900) · Бронза (901–1400) · Серебро (1401–2000) · Золото (2001–2800) · Кристалл (2801–3700) · Легенда (3701–4500) · Архонт (4501–4900) · Божество (4901–5000).

**Атлетический ранг** — по среднему Сила+Выносливость, скорректирован физ-тестами:
Новичок · Любитель · Атлет · Спортсмен · Мастер · Элита · Чемпион · Легенда · Божество.
Повышение требует не только XP, но и пересдачи физ-тестов.

Систему рангов вводим после полировки базовой RPG (см. roadmap.md).

## Визуал (целевой)

Радарная диаграмма (Recharts `RadarChart`) + ранги + прогресс-бар до следующего ранга. Архив: график роста каждого стата по неделям (нужна таблица `stats_history` — её ещё нет).
