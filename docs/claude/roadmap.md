# Роадмап

> Назад: [CLAUDE.md](../../CLAUDE.md)
> Принцип: **сначала крошечная рабочая версия, потом по одной функции.** Не строить модули параллельно.

## MVP — этапы в боте (текущая фаза)

| Этап | Что | Статус |
|---|---|---|
| Скелет | FastAPI+aiogram, /health, вебхук, /start, БД | ✅ |
| 1 · Вода | `/water`, кнопки, +2 XP → Здоровье | ✅ |
| 2 · Ритуалы | `/addritual`, `/rituals`, отметка, стрик, +2 XP → Дисциплина | ✅ |
| **3 · Задачи** | добавить/список/выполнить, +XP → Дисциплина | **следующий** |
| 4 · Питание | ручной ввод КБЖУ, +XP → Питание | планируется |
| 5 · Дашборд | `/today` + `/stats` (8 статов, уровень) | планируется |
| 6 · Кнопки | постоянный Reply Keyboard (`/tasks /water …`) | планируется |

**MVP готов, когда:** ежедневно ведёшь через бота воду, ритуалы, задачи, питание и видишь рост XP.

## После MVP

**Фаза 2 — Mini App.** React+Vite+TS+@telegram-apps/sdk. Хостинг фронта статикой (бесплатно). Дизайн-система из `conventions.md`. Экраны: дашборд, 4 MVP-модуля, RPG-паутинка. Авторизация: фронт→бэкенд по REST, бэкенд проверяет initData (без полного JWT-моста).

**Фаза 3 — остальные модули по одному.** Каждый = миграция + хендлер + экран: Дневник, Сон, Финансы, Цели, Чтение/учёба, Встречи, Идеи, Inbox, Замеры, Колесо баланса, Дайджест (пока без AI).

**Фаза 4 — тело и AI.** Тренировки (лог + шаблоны). AI на Gemini: фото→КБЖУ, голос→интент, AI-тренер. См. `ai-layer.md` (сначала решить узлы по whisper/Storage).

**Фаза 5 — автоматизация.** Уведомления + дайджест. Сначала решить планировщик: внешний cron (cron-job.org / Supabase pg_cron) дёргает эндпоинт.

**Фаза 6 — наружу (опционально).** Мультипользователь: мост initData→JWT, онбординг.

## Узлы — решить ПЕРЕД фазой

1. **Планировщик на free Render** (Фаза 5) — сервис спит, APScheduler в процессе не сработает → внешний cron.
2. **faster-whisper и 512MB** (Фаза 4) — модель medium не влезет.
3. **Лимит Supabase Storage ~1GB** (Фаза 4) — политика автоудаления медиа.
4. **Мост initData→JWT** (Фаза 6) — на MVP не нужен.

## Честно про сроки

Оригинал обещал ~17 недель. Реально для новичка в одиночку — **месяцы**. Это нормально. Темп: одна вещь за раз, до рабочего состояния, потом следующая. Лучше медленно и доделать, чем быстро и забросить.
