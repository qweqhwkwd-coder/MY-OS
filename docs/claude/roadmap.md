## После MVP — фактическое состояние

**Фаза 2 — Mini App.** 🟡 Запущена, не «после», а **параллельно** с расширением
модулей через бота (вопреки принципу «по одному» выше — так сложилось по факту).
React 19 + Vite + Tailwind v4 (не TS-стек из старого плана — TanStack
Query/Zustand/shadcn/ui не внедрены), статика на GitHub Pages. Экраны: Today,
Water, Rituals, Tasks, Food + общий профиль (ранг/HP/стрик). Авторизация —
initData проверяется бэкендом (без JWT-моста), как и планировалось. Сетка
навигации (`NavGrid`) уже показывает плитки для 7 дальнейших модулей как
заблокированные — экраны для них не сделаны.

**Фаза 3 — остальные модули.** 🟡 Почти полностью сделана **через бота**
(не через Mini App, как подразумевал план): Дневник ✅, Сон ✅, Финансы ✅,
Цели ✅, Встречи ✅, Идеи ✅, Inbox ✅, Замеры ✅, Колесо баланса ✅,
Дайджест ✅ (без AI). Не сделано: Чтение/учёба (нет источника XP в
Интеллект), и ни один из этих модулей не имеет экрана в Mini App.

**Фаза 4 — тело и AI.** 🟡 частично. Тренировки: ручной лог + авто-определение
типа по ключевым словам — ✅. Шаблоны активностей, AI-тренер — ❌. AI на
Gemini (фото→КБЖУ, голос→интент) — ❌, не начато. См. `ai-layer.md` (узлы
по whisper/Storage ещё актуальны, не решены).

**Фаза 5 — автоматизация.** ❌ не начата. Планировщик/уведомления/дайджест-по-расписанию
не реализованы; `/digest` есть только по запросу.

**Фаза 6 — наружу (опционально).** ❌ не начата. Мультипользователь:
мост initData→JWT не реализован.

## Узлы — решить ПЕРЕД фазой (актуальность не изменилась)

1. **Планировщик на free Render** (Фаза 5) — сервис спит, APScheduler в процессе не сработает → внешний cron.
2. **faster-whisper и 512MB** (Фаза 4) — модель medium не влезет.
3. **Лимит Supabase Storage ~1GB** (Фаза 4) — политика автоудаления медиа.
4. **Мост initData→JWT** (Фаза 6) — на MVP не нужен, на одном юзере с Mini App тоже пока не нужен.

## Реальный следующий шаг

Решить осознанно, а не по умолчанию: либо **достраивать Mini App** под уже
готовые бэкенд-модули (Дневник/Сон/Финансы/Цели/...), либо **подключать AI**
(Фаза 4). Сейчас бэкенд обгоняет фронт по фичам, а фронт обгоняет план по
дизайну — стоит сначала свести это к одному списку, прежде чем добавлять
что-то новое.

## Честно про сроки

Оригинал обещал ~17 недель. Реально для новичка в одиночку — **месяцы**. Это нормально. Темп: одна вещь за раз, до рабочего состояния, потом следующая. Лучше медленно и доделать, чем быстро и забросить.
