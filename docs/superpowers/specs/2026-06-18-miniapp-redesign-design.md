```

├── WelcomeScreen       — показується при першому відкритті дня
├── SysBar              — постійна чорна полоса зверху
```

## 3. Welcome Screen

- Чорний фон (`#1a1a1a`)
- Логотип `MY-OS · v1.0` малим шрифтом зверху
- Цитата стоїків — **рандомна щодня** з набору ~10 цитат (Марк Аврелій, Епіктет, Сенека, Франциск)
- Дата + стрік внизу
- Кнопка `Відкрити систему →` — біла, повна ширина
- Показується **один раз на день** (localStorage: `last_welcome_date`)

## 4. SysBar (постійна)

Чорна полоса зверху на всіх екранах:

```
[HP: 78] | [LV 3] | [+14 XP] | [5🔥] | [⚔️ кнопка профілю]
```

- HP — зелений/жовтий/червоний залежно від значення

- ⚔️ — кнопка що відкриває ProfileModal
- Всі значення — IBM Plex Mono

## 6. Сторінка "Сьогодні" (01)

**Блок "Швидко додати"** (4 кнопки в ряд):
- 💧 Воду → відкриває мінімальний modal: кнопки +250 / +500 / +1000
- ✅ Задачу → modal: текстове поле + кнопка Додати
- 📝 Нотатку → modal: textarea + Зберегти (йде в inbox/ideas)
- 🍽 Їжу → modal: назва + ккал + Додати

**Дайджест дня** (hairline rows):
- Вода: прогрес-бар + `1240 / 2000 мл`
- Ритуали: прогрес-бар + `3/4`
- Завдання: `2 виконано`
- Калорії: прогрес-бар + `1100 / 2000`

---

## 7. Profile Modal

Відкривається поверх поточної сторінки (slide-up або overlay).

**Секції зверху вниз:**

1. **Герой:** аватар-емодзі + ім'я + `LV N` badge
2. **Ранг + відсоток:**
   - Ліворуч: назва рангу великим шрифтом + прогрес-бар до наступного
   - Праворуч: велике число `42%` + підпис "ти кращий за X% людей"
3. **HP:** зелений бар 0–100 + підказка що вплинуло вчора
4. **XP:** індиго бар + "залишилось N xp до рівня N+1"
5. **Характеристики:** 8 рядків, кожен: назва + мінібар + `lvN·Nxp`

## 8. HP Механіка

**Формула (перераховується при кожному `/api/profile` запиті):**
```
hp = round(
  water_pct  * 0.30 +
  ritual_pct * 0.40 +
  sleep_pct  * 0.20 +
  food_pct   * 0.10
)
```
де `pct` = відсоток виконання за останні 3 дні (rolling average).

**Зберігання:** поле `hp INTEGER DEFAULT 50` в `user_stats`.  
**Міграція:** `ALTER TABLE user_stats ADD COLUMN IF NOT EXISTS hp INTEGER DEFAULT 50;`

## 10. Що НЕ входить в цей спринт

- Сторінки заблокованих модулів (Сон, Фінанси тощо)
